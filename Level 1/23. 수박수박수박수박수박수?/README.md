## 문제 설명

길이가 `n`이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수 `solution`을 완성하세요.  
예를 들어, `n`이 4이면 `"수박수박"`을 리턴하고, 3이면 `"수박수"`를 리턴하면 됩니다.

### 제한 조건
- `n`은 길이 10,000 이하인 자연수입니다.

## 입출력 예

| n  | return     |
|----|------------|
| 3  | "수박수"   |
| 4  | "수박수박" |

## 입출력 예 설명

### 입출력 예 #1
`n`이 3이면 "수박수"가 출력됩니다. `"수박"` 패턴을 반복하여 길이를 맞추면 `"수박수"`가 됩니다.

### 입출력 예 #2
`n`이 4이면 "수박수박"이 출력됩니다. `"수박"` 패턴을 반복하여 길이를 맞추면 `"수박수박"`이 됩니다.
