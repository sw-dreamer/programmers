## 문제 설명

정수 `num`이 짝수일 경우 `"Even"`을 반환하고 홀수인 경우 `"Odd"`를 반환하는 함수, `solution`을 완성해주세요.

### 제한 조건
- `num`은 `int` 범위의 정수입니다.
- 0은 짝수입니다.

## 입출력 예

| num | return |
|-----|--------|
| 3   | "Odd"  |
| 4   | "Even" |

## 입출력 예 설명

### 입출력 예 #1
3은 홀수이므로 "Odd"를 반환합니다.

### 입출력 예 #2
4는 짝수이므로 "Even"을 반환합니다.
