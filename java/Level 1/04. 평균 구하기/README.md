## 문제 설명

정수를 담고 있는 배열 `arr`의 평균값을 `return`하는 함수, `solution`을 완성해보세요.

### 제한사항
- `arr`은 길이 1 이상, 100 이하인 배열입니다.
- `arr`의 원소는 -10,000 이상 10,000 이하인 정수입니다.

## 입출력 예

| arr        | return |
|------------|--------|
| [1, 2, 3, 4] | 2.5    |
| [5, 5]      | 5      |

## 입출력 예 설명

### 입출력 예 #1
배열 `[1, 2, 3, 4]`의 평균은 `(1 + 2 + 3 + 4) / 4 = 2.5`입니다.

### 입출력 예 #2
배열 `[5, 5]`의 평균은 `(5 + 5) / 2 = 5`입니다.
