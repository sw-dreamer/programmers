## 문제 설명

문자열 `s`의 길이가 4 혹은 6이고, 숫자로만 구성돼 있는지 확인해주는 함수, `solution`을 완성하세요.  
예를 들어 `s`가 `"a234"`이면 `False`를 리턴하고 `"1234"`라면 `True`를 리턴하면 됩니다.

### 제한 사항
- `s`는 길이 1 이상, 길이 8 이하인 문자열입니다.
- `s`는 영문 알파벳 대소문자 또는 0부터 9까지 숫자로 이루어져 있습니다.

### 입출력 예

| s      | return |
|--------|--------|
| "a234" | false  |
| "1234" | true   |
