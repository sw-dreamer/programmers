## 문제 설명

단어 `s`의 가운데 글자를 반환하는 함수 `solution`을 만들어 보세요.  
단어의 길이가 짝수라면 가운데 두 글자를 반환하면 됩니다.

### 제한사항
- `s`는 길이가 1 이상, 100 이하인 문자열입니다.

## 입출력 예

| s      | return |
|--------|--------|
| "abcde" | "c"    |
| "qwer"  | "we"   |

## 입출력 예 설명

### 입출력 예 #1
단어 `abcde`의 가운데 글자는 `c`입니다.

### 입출력 예 #2
단어 `qwer`의 가운데 두 글자는 `we`입니다.
